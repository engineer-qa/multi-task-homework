# Барбершоп на Semaphore

Реализация барбершопа в многопоточном режиме с использованием двух семафоров:
один ограничивает доступ к креслу барбера, второй - количество клиентов в зоне ожидания

Используемые элементы
Semaphore barber = new Semaphore(1, 1);

Ограничивает количество клиентов, которых барбер может обслуживать одновременно (1 поток)

Semaphore waitingChairs = new Semaphore(3, 3);

Ограничивает количество клиентов, которые могут ожидать своей очереди (3 потока)

Основная логика

Клиент приходит и пытается занять стул ожидания:

waitingChairs.WaitOne(0)
Если нет свободного места — клиент уходит

Если место найдено — клиент «садится» и ждёт освобождения барбера

Когда барбер доступен:

клиент занимает кресло (barber.WaitOne()),

освобождает стул ожидания (waitingChairs.Release()),

выполняется имитация стрижки через Thread.Sleep

После завершения обслуживания клиент освобождает кресло (barber.Release())
